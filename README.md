## 简介

- `@z-xuanyu/utils` 前端开发常用的工具函数

## 安装

```bash
# npm
$ npm i @z-xuanyu/utils

# yarn
$ yarn add @z-xuanyu/utils

# pnpm
$ pnpm add @z-xuanyu/utils
```

## 使用

```typescript
import {isNumber} from "@z-xuanyu/utils";

const num = 3;
// 判断num是否位number类型
isNumber(num);
```
